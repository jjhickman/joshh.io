# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-02-24

### Added

- Initial project scaffold with Vite, React 19, TypeScript, and Tailwind CSS v4
- Layout shell: Header with navigation and dark mode toggle, Footer with social links
- Home page with hero section, featured projects, and "what I do" cards
- About page with bio, experience timeline, and skills visualization
- Portfolio page with project cards and technology tag filtering
- Blog page (placeholder, infrastructure ready)
- Music work-in-progress page
- Contact page with social links
- Structured data layer (experience, projects, skills)
- Custom hooks: useTheme (dark/light mode), useMediaQuery
- Reusable UI components: Button, Card, Badge, Section
- Shared components: ProjectCard, ExperienceTimeline, SkillBadges, SocialLinks
- Framer Motion page transitions and scroll animations
- ESLint flat config with strict TypeScript rules
- Vitest + Testing Library test setup
- AWS Amplify deployment configuration
- GitHub Actions CI workflow
- Documentation: AGENTS.md, README.md, ARCHITECTURE.md, DEPLOYMENT.md, DESIGN.md
