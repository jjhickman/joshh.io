# garbanzo-bot

AI chat operations platform for communities. Multi-provider LLM routing with 500+ tests, 162+ commits, and Docker-first deployment.

Built with TypeScript and Node.js, garbanzo-bot is a comprehensive chat operations platform designed for community management. It features multi-provider LLM routing, configurable personalities, rate limiting, and extensive test coverage.

## Key Features

- Multi-provider LLM routing (OpenAI, Anthropic, and more)
- Configurable bot personalities
- Rate limiting and abuse prevention
- Docker-first deployment philosophy
- 500+ tests with Vitest
- Apache-2.0 licensed
