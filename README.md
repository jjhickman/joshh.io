# joshh.io

Music-first React SPA for Josh Hickman and IN CASE OF EMERGENCY.

## Local development

Requires Node 20 or newer.

```sh
npm install
npm run dev
```

Mutable records and copy live in `src/content`; replaceable placeholder artwork lives in `src/assets/images`. Run `npm run check`, `npm run build`, and `npm run check:bundle` before handing off a change.
